CSS: каскадные таблицы стилей

Cascading Style Sheets (CSS) — это набор иерархических правил (таблиц стилей), используемых для представления внешнего вида документа, написанного на HTML или XML (включая различные языки XML, такие как SVG и XHTML). CSS должен отображаться на экране, на бумаге, голосом или с использованием других media средств.

CSS является одним из основных языков свободной веб-разработки, который стандартизован спецификацией W3C. Стандарт CSS делится на уровни:

  • CSS1 в настоящее время устарел
  • CSS2.1 — рекомендуется для применения
  • CSS3, разбитый на более мелкие модули, развивается на пути стандартизации
